# FitZone Gym Management System

Hệ thống quản lý phòng tập gym chuyên nghiệp.

## Tính năng

- Dashboard tổng quan
- Quản lý hội viên
- Quản lý gói tập
- Quản lý huấn luyện viên
- Lịch tập
- Check-in
- Quản lý thiết bị
- Hóa đơn
- Báo cáo thống kê
- Cài đặt hệ thống

## Công nghệ

- HTML5
- CSS3
- JavaScript (Vanilla)
- LocalStorage

## Cài đặt

1. Clone repository
2. Mở file `admin/index.html` trong trình duyệt
